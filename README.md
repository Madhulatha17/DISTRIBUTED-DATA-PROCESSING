# Project Overview

This project demonstrates **distributed data processing using Apache Spark**.  
It performs large-scale data analysis using Spark DataFrame operations such as filtering, grouping, and aggregations.


## Objective

To process and analyze large datasets using Apache Spark by implementing:

- Filtering operations  
- GroupBy aggregations  
- Statistical computations  
- Distributed processing concepts  


## Features

- Load large dataset using Spark  
- Filter specific categories  
- Perform group-based aggregations  
- Compute summary statistics  
- Distributed computation using Spark engine  



## Operations Performed

### 1. Filtering
Extracts only selected category data (e.g., Electronics)

### 2. Grouping
Groups data by category and product

### 3. Aggregation
- Total quantity sold  
- Average price  
- Product-wise sales  



##  Dataset Description

| Column | Description |
|--------|-------------|
| id | Unique ID |
| product | Product Name |
| category | Product Category |
| quantity | Units sold |
| price | Product Price |
